SERVER PERFORMANCE MONITORING SCRIPT

Project Url: https://github.com/Ayeebeedeee/server-stats.sh/blob/main/code

Project Description:
This project is a Bash script that monitors and displays basic server performance statistics. It helps in understanding how system resources like CPU, memory, and disk are being used.

Features:

* Displays total CPU usage (user, system, idle, total)
* Displays memory usage (total, used, free, percentage)
* Displays disk usage (total, used, free, percentage)
* Shows top 5 processes based on CPU usage
* Shows top 5 processes based on memory usage

Technologies Used:

* Bash Scripting
* Linux commands: top, free, df, ps, awk, tr

How to Run:

1. Make the script executable:
   chmod +x server_performance.sh

2. Run the script:
   ./server_performance.sh

Output:
The script prints a formatted report showing system performance details including CPU, memory, disk usage, and running processes.

Purpose:
This project is useful for beginners to understand Linux system monitoring and Bash scripting. It is also helpful for basic DevOps and system administration tasks.
